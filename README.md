# 🗺️ 人生地图 · life-map

> 上传你走过的照片,自动落在地图上,标记你的人生足迹。

一个 **纯前端、本地存储、零成本** 的个人足迹地图:照片按 EXIF 定位自动钉在地图上,没定位的手动补,每处写下当时的故事。**数据全在你自己的浏览器里,不上传任何服务器。**

## ✨ 功能

- 📸 **批量上传** —— 多选 / 选整个文件夹 / 拖拽,自动读 EXIF GPS 落点
- 🗺️ **地图足迹** —— 照片缩略图直接作为地图钉子(Esri 街道图 / 卫星图可切换)
- 📍 **手动补点** —— 没有 GPS 的照片,点一下地图即可放置
- 📝 **地点故事** —— 点开任意钉子,写下当时的备注
- 💾 **本地存储** —— 数据存在浏览器 IndexedDB,刷新 / 重开都在,离线可用
- 📤 **导出 / 导入** —— 一键备份为 JSON(含照片),换设备可恢复

## 🧱 技术栈

- **Vue 3 + Vite**
- **Leaflet** + **Esri** 瓦片(WGS-84,国内可访问)
- **Dexie**(IndexedDB)本地存储
- **exifr** 解析照片 EXIF(GPS / 拍摄时间)
- 无后端、无 AI、无追踪

## 🚀 本地运行

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # 产物在 dist/
```

## 📌 使用说明 / 已知限制

- **照片要有 GPS 才能自动落点**:手机相册原图(拍摄时开了定位)最可靠;**微信收到的、截图、社交平台下载的照片,定位信息通常已被抹除**,会进"待定位",手动补即可。
- **iPhone HEIC**:能读到定位、能落点,但浏览器暂时无法直接显示 HEIC 缩略图(后续支持)。用 JPG 最稳。
- **数据在本地浏览器**:清除浏览器数据会丢,记得用「导出」定期备份。
- 国内 OpenStreetMap 瓦片常连不上,故默认使用 Esri 底图。
