# 🗺️ 人生地图 · life-map

> 上传你走过的照片,自动落在地图上,标记你的人生足迹。

一个 **纯前端、本地存储、零成本** 的个人足迹地图:照片按 EXIF 定位自动钉在地图上,没定位的手动补,每处写下当时的故事;还能回放旅程、点亮去过的省份、生成分享海报。**数据全在你自己的浏览器里,不上传任何服务器。**

## ✨ 功能

**基础**
- 📸 **批量上传** —— 多选 / 选整个文件夹 / 拖拽,自动读 EXIF GPS 落点(带进度)
- 🗺️ **照片地图** —— 照片缩略图直接作为地图钉子
- 🧩 **地点聚合** —— 邻近足迹聚成气泡(用照片当封面 + 数字角标),放大自动散开
- 📍 **手动补点** —— 没有 GPS 的照片,点一下地图即可放置
- 📝 **地点故事** —— 点开钉子写备注,点大图进全屏查看

**视觉**
- 💡 **点亮足迹** —— 去过的地方在深色地图上罩着青蓝暖光、呼吸般明灭,像一盏盏被点亮的灯;新落点会「叮」地炸开一圈光晕
- 🌗 **深 / 浅主题切换** —— 一键切换深色 / 浅色地图与界面
- 🀄 **中英文地图** —— 英文(Esri)/ 中文(高德)一键切换,自动 WGS-84↔GCJ-02 坐标纠偏
- ✨ **玻璃质感 UI + 平滑飞行动画** —— 毛玻璃面板、`flyTo` 过渡、钉子落下/悬停动效

**探索与回顾**
- ✈️ **旅程回放** —— 按「地区」把足迹连成**飞机航线**,一架飞机沿弧线航迹一路飞过、镜头跟随,重温你的旅程
- 🕐 **时间线回顾** —— 按年份筛选,只看某一年的足迹
- 🧭 **行程聚类** —— 相近时间的足迹自动归成一段段「旅程」,一键回看
- 📊 **里程看板** —— 按时间顺序算总里程 + 城市 / 足迹 / 照片 / 最常去 / 时间跨度
- 🗺️ **点亮城市** —— 只高亮你真正去过的**地级市**(不是整省染色),按需加载城市边界
- 🖼️ **全屏 lightbox** —— 大图浏览,键盘 ←/→ 或按钮切换同地点照片
- ▦ **相册网格** —— 所有照片网格视图,点开即看
- 🔍 **搜索** —— 按城市 / 备注关键词跳转到对应足迹
- 🎁 **去年今日** —— 往年同一天拍的照片,自动提醒回忆
- 🖼️ **路线海报** —— 一键生成足迹路线海报(按时间连线 + 里程统计),PNG 可分享

**数据**
- 🏙️ **城市统计** —— 自动反查照片所在城市
- 📱 **HEIC 支持** —— iPhone 原图(.heic)自动转码显示
- 💾 **本地存储(持久化)** · 📤 **ZIP 导出 / 导入**(兼容旧版 JSON) · 🧹 **一键清空**
- 📱 **移动端适配** —— 手机浏览器也能顺手用

## 🧱 技术栈

- **Vue 3 + Vite**
- **Leaflet** + **leaflet.markercluster** + **Esri** 灰底瓦片(WGS-84,国内可访问)
- **Dexie**(IndexedDB)本地存储
- **exifr** 解析 EXIF(GPS / 拍摄时间)
- **heic2any / jszip**(按需懒加载)HEIC 转码 / ZIP 备份 · **BigDataCloud** 反查城市 · **高德** 中文底图
- 无后端、无 AI、无追踪

## 🚀 本地运行

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # 产物在 dist/
```

## 📌 使用说明 / 已知限制

- **照片要有 GPS 才能自动落点**:手机相册原图(拍摄时开了定位)最可靠;**微信收到的、截图、社交平台下载的照片,定位信息通常已被抹除**,会进「待定位」,手动补即可。
- **数据在本地浏览器**:清除浏览器数据会丢,记得用「导出」定期备份。
- 国内 OpenStreetMap / 部分地理服务常连不上,故底图用 Esri、城市反查用 BigDataCloud、省界用阿里云 DataV。
